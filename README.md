# PuFU
Personal Flying Umbrella

An attempt to design and build a minimalistic flying vehicle.

## Goals

- Lightweight,
- Stability,
- Long autonomy (15 minutes at least),
- Easy to carry,
- Easy to fly with,
- Easy to take off,
- Easy to land with.
